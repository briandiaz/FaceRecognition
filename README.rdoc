FaceRecognition
====================
This is a rails web app that consumes the Face API. It detects your face, eyes, mouth, nose, smile rate and your confidence rate.

Running the app
-----------
In your command line just do
rake db:migrate
and run the server
rails s

Enjoy it!
If you want to colaborate just ask.


Resources
-----------

Frameworks/API's:
+ Face (https://www.mashape.com/lambda/face#!documentation)
+ Boostrap 3 (https://github.com/twbs/bootstrap)
